## 麒麟-昇腾显卡-部署视频分析算法

### 一、环境搭建

```
1.拉取镜像
	docker pull swr.cn-south-1.myhuaweicloud.com/ascendhub/ubuntu:24.04-base:latest

2.生成容器
    docker run -itd \
      --name video-analysis \
      --privileged \
      --network host \
      --restart unless-stopped \
      -v /usr/local/Ascend:/usr/local/Ascend:rw \
      -v /usr/local/dcmi:/usr/local/dcmi:ro \
      -v /etc/ascend_install.info:/etc/ascend_install.info:ro \
      -v /sys/kernel/debug:/sys/kernel/debug:rw \
      --device=/dev/davinci0 \
      --device=/dev/davinci2 \
      --device=/dev/davinci4 \
      --device=/dev/davinci6 \
      --device=/dev/davinci_manager \
      --device=/dev/hisi_hdc \
      -v /home/scv/video_analysis/wdd:/workspace/wdd \
      swr.cn-south-1.myhuaweicloud.com/ascendhub/ubuntu:24.04-base \
      /bin/bash
      
3.安装依赖
	apt update
    apt install -y software-properties-common
    add-apt-repository universe
    apt update
    apt install -y gcc make net-tools python3 python3-dev python3-pip python3-numpy python3-decorator python3-pyparsing python3-wheel python3-setuptools python3-sympy python3-scipy python3-matplotlib python3-psutil

4.安装CANN
	./Ascend-cann-toolkit_8.5.0_linux-aarch64.run --install
	./Ascend-cann-310p-ops_8.5.0_linux-aarch64.run --install
```

### 二、模型转换

```
1.模型文件*.pt转换成*.onnx
	yolo export model=D:\company_project\ESI200\yolo_to_onnx_c++\people_track_model\suitcase.pt format=onnx imgsz=640 opset=12 dynamic=True simplify=True

2.模型文件*.onnx转换成*.om
	atc \
    --model=suitcase.onnx \
    --framework=5 \
    --output=suitcase_dynamic_om \
    --input_shape="images:-1,3,640,640" \  # ✅ -1 = 动态Batch
    --dynamic_batch_size="1,2,4,8,16,32,48,64" \  # ✅ 支持 1~96 路
    --input_format=NCHW \
    --soc_version=Ascend310P3 \
    --insert_op_conf=aipp_yuv_hardware.cfg \
    --precision_mode=allow_mix_precision

```
