
### 三、修改配置文件

```
cd /opt/diag/   # 开机后进入的目录
./cfg.sh        # 执行脚本
	弹框选择输入：2，之后一直回车就行；
sudo vim /etc/gdm3/custom.conf
	1.将“AutomaticLoginEnable=True”改为“AutomaticLoginEnable=False”
	2.将“AutomaticLogin=root”改为“AutomaticLogin=orin”
sudo reboot
```
