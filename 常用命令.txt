3.进程查询：
	ps -ef|grep python
# 用来搜索正在运行的Python进程。
ps: 显示进程的命令。
-e: 选择所有进程。
-f: 显示完整格式的信息。

7.后台启动某个服务
	nohup python 脚本名 &
	例子：
		nohup python my_script.py &
		nohup python my_script.py > my_output.log 2>&1 &
	注释：
		nohup：这个命令使得即使你关闭了终端，命令也会继续运行。
		python4 my_script.py：这是你想要运行的Python命令。
		&：这个符号将命令放到后台执行。

9.ubantu系统上安装python
第一步：进入终端后输入以下：
	sudo apt update
第二步：安装PPA需要的软件源:
	sudo apt install software-properties-common -y
第三步：添加名为deadsnake的PPA源:
	sudo add-apt-repository ppa:deadsnakes/ppa
第四步：安装python3.10:
	sudo apt install python3.10
第五步：安装好后，通过以下命令验证:
	python3.12 --version
第六步：如果你想要python命令指向Python 3.10，你可以在你的shell配置文件中（如.bashrc或.bash_profile）添加：
	alias python='/usr/bin/python3.10'

注意：
	.bashrc 文件通常位于用户的家目录中，即 /home/用户名/。例如，如果你的用户名是 john，那么 .bashrc 文件将位于 /home/john/.bashrc。


10.如果在ubantu系统上，新创建的用户不具备使用sudo的权限，该怎么操作？
第一步：切换至root或可以使用 sudo 的用户账户；
第二步：输入命令：
					sudo visudo
第三步：编辑 sudoers 文件，在这个文件中添加一行来允许 wdd 用户使用 sudo。
		wdd ALL=(ALL:ALL) ALL
第四步：保存退出。


11.查看ubantu系统的摄像头设备文件
第一步：查看设备文件：
						ls /dev/video*
第二步：需要确保用户具备使用设备文件的权限，将您的用户添加到video组（请替换$USER为您的用户名）：
		sudo usermod -aG video $USER
第三步：重新登录。

15.连接远程ubantu服务器，用户和密码正确，terminal怎么连接不上
方法一：执行sudo apt-get install openssh-server来安装SSH服务，安装完成后该服务通常会自动启用。
方法二：使用sudo ufw status查看防火墙状态，并使用sudo ufw allow ssh或sudo ufw allow 22来允许SSH连接。
